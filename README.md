# DISK

## Installation

1. Clone the repository [https://github.com/IKCAP/DISK.git](https://github.com/IKCAP/DISK.git)

```bash
git clone https://github.com/IKCAP/DISK.git
```

2. Setup

    2.1 For Development

    ```bash
    python setup.py develop
    ```

    2.1 For Installation
    ```bash
    python setup.py install
    ```
